# Endspace Theme For NotionNext

[![Awesome Arknights Endfield](https://github.com/palmcivet/awesome-arknights-endfield/blob/main/assets/badge.svg?raw=true)](https://github.com/palmcivet/awesome-arknights-endfield)

> 『アークナイツ：エンドフィールド』の視覚的な美学にインスパイアされたNotionNextテーマ。

**Endspace**は、個人ブログに未来主義とインダストリアルSFの独特な雰囲気をもたらすことを目指した、非常にスタイリッシュなNotionNextテーマです。そのUIデザインは、『アークナイツ：エンドフィールド』公式サイトのビジュアルスタイルに深く敬意を表しており、厳格なグリッドシステム、技術的なデータオーバーレイ、そして識別性の高い配色を特徴としています。

## ✨ 特徴

- **インダストリアルSFデザイン**：厳格なレイアウト、等幅フォント、およびテクノロジー感のある装飾要素を採用。
- **没入型ローディング体験**：システム初期化ステップとスキャンアニメーションを含むカスタムローディングシーケンス。
- **統合メディア再生**：音楽プレーヤーコンポーネントを内蔵。
- **レスポンシブレイアウト**：デスクトップおよびモバイルでの閲覧体験向けに深く最適化されています。
- **豊富な設定項目**：色、アニメーション、表示要素をカスタマイズするための広範なオプションを提供。

## 🚀 インストールと使用法

本テーマのソースコードは `theme` ブランチにあります。

1.  **テーマのダウンロード**：本リポジトリの `theme` ブランチからすべてのファイルをダウンロードしてください。
2.  **インストール**：ダウンロードしたファイルをNotionNextプロジェクトの `themes` フォルダに配置し、フォルダ名を `endspace` に変更してください（パスは `themes/endspace` となります）。
3.  **有効化**：NotionNextの環境変数または設定ファイルを変更し、`THEME` を `endspace` に設定することで有効になります。

## 💡 よくある質問 (Q&A)

### NotionNextとは何ですか？

NotionNextは、Notion APIに基づく静的ブログジェネレーターであり、ユーザーがNotionデータベースを通じて直接ブログコンテンツを作成および管理できるようにします。

👉 [NotionNext 公式ドキュメント](https://docs.tangly1024.com/)

### テーマの設定方法は？

テーマの設定やカスタマイズ方法については、公式ドキュメントを参照してください：

👉 [カスタムテーマガイド](https://docs.tangly1024.com/article/your-own-theme)

### なぜEndspaceはNotionNextの公式テーマに含まれていないのですか？

カスタマイズ性が非常に高く、公式に貢献するには多くの修正が必要となるため、自分用として使用することにしました。

👉 [プレビュー](https://cloud09.space)
