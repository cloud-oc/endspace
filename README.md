# Endspace Theme For NotionNext

[![Awesome Arknights Endfield](https://github.com/palmcivet/awesome-arknights-endfield/blob/main/assets/badge.svg?raw=true)](https://github.com/palmcivet/awesome-arknights-endfield)

> 受《明日方舟：终末地》视觉美学启发的 NotionNext 主题。

**Endspace** 是一款风格强烈的 NotionNext 主题，致力于为您的个人博客赋予未来主义与工业科幻的独特气质。其 UI 设计深度致敬了[《明日方舟：终末地》官方网站](https://endfield.hypergryph.com/)的视觉风格。

## ✨ 特性

- **工业科幻设计**：采用严谨的布局、等宽字体及科技感装饰元素。
- **沉浸式加载体验**：包含系统初始化步骤与扫描动效的自定义加载序列。
- **集成媒体播放**：内置音乐播放器组件。
- **响应式布局**：针对桌面端与移动端阅读体验进行了深度优化。
- **丰富的配置项**：提供广泛的选项以自定义颜色、动画及显示元素。

## � 安装与使用

本主题源代码位于 `theme` 分支。

1.  **下载主题**：请下载本仓库 `theme` 分支的所有文件。
2.  **安装**：将下载的文件放置于 NotionNext 项目的 `themes` 文件夹下，并重命名为 `endspace`（即路径应为 `themes/endspace`）。
3.  **启用**：修改 NotionNext 的环境变量或配置文件，将 `THEME` 设置为 `endspace` 即可生效。

## 💡 常见问题 (Q&A)

### NotionNext 是什么？

NotionNext 是一个基于 Notion API 的静态博客生成器，允许用户直接通过 Notion 数据库创建和管理博客内容。

👉 [NotionNext 官方文档](https://docs.tangly1024.com/)

### 如何配置主题？

关于如何配置和自定义您的主题，请参考官方文档：

👉 [自定义主题指南](https://docs.tangly1024.com/article/your-own-theme)

### Endspace为什么没有收录在 NotionNext 的主题中？

个性化程度过高，贡献给官方会有不少要修改的，索性就自己用了。

👉 [效果预览](https://cloud09.space)
