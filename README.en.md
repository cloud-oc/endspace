# Endspace Theme For NotionNext

[![Awesome Arknights Endfield](https://github.com/palmcivet/awesome-arknights-endfield/blob/main/assets/badge.svg?raw=true)](https://github.com/palmcivet/awesome-arknights-endfield)

> A NotionNext theme inspired by the visual aesthetics of *Arknights: Endfield*.

**Endspace** is a highly stylized NotionNext theme designed to bring a unique futuristic and industrial sci-fi atmosphere to your personal blog. Its UI design pays deep homage to the visual style of the official *Arknights: Endfield* website, featuring a rigorous grid system, technical data overlays, and a distinctive color palette.

## âœ¨ Features

- **Industrial Sci-Fi Design**: Rigorous layout, mono-spaced typography, and tech-inspired decorative elements.
- **Immersive Loading Experience**: Custom loading sequence including system initialization steps and scanning effects.
- **Integrated Media Playback**: Built-in music player component.
- **Responsive Layout**: Deeply optimized for both desktop and mobile reading experiences.
- **Rich Configuration Options**: Extensive settings to customize colors, animations, and display elements.

## ðŸš€ Installation & Usage

The source code for this theme is located in the `theme` branch.

1.  **Download Theme**: Please download all files from the `theme` branch of this repository.
2.  **Install**: Place the downloaded files into the `themes` folder of your NotionNext project and rename the folder to `endspace` (so the path should be `themes/endspace`).
3.  **Enable**: Modify the NotionNext environment variables or configuration file and set `THEME` to `endspace`.
